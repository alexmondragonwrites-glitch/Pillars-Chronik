# Die Gesänge der Serin Ashwyn

Ein erzählerisches Webtagebuch zu unserem *Pillars of Eternity*-Durchlauf.

Im Mittelpunkt steht **Serin Ashwyn**, eine mondgottgleiche Sängerin und ehemalige Lehrerin aus Aedyr. Die Chronik bewahrt ihre Entscheidungen, Zweifel, Verluste und ihre Entwicklung zur Wächterin.

## Website

Die statische Website liegt direkt im Repository und wird über GitHub Pages veröffentlicht.

- `index.html` bildet das Webtagebuch ab.
- `data/entries.json` enthält die sichtbaren Chronikeinträge und Entscheidungen.
- `data/serin.json` enthält das Charakterprofil.
- `content/chronik/` bewahrt die Kapitel zusätzlich als Markdown.
- `content/charaktere/` enthält die kanonischen Charakterdokumente.
- `assets/` enthält die visuellen Referenzen.

## Neue Kapitel ergänzen

1. Einen neuen Eintrag in `data/entries.json` ergänzen.
2. Das Kapitel zusätzlich in `content/chronik/` als Markdown archivieren.
3. Änderungen nach `main` übertragen. Der Pages-Workflow veröffentlicht die Website automatisch.

## Kanonischer Ausgangspunkt

Serin wollte Heodan in den Ruinen von Cilant Lîs die Möglichkeit geben, sich selbst zu befreien. Er starb durch spektrale Energie. Dieser Verlust ist kein zurückgesetzter Spielstand, sondern der erste prägende Riss in Serins Weltbild:

> Auch Nichtstun ist eine Entscheidung. Manchmal trägt Zurückhaltung dieselbe Schuld wie eine falsch geführte Waffe.
