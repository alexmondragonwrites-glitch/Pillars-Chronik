# Dialog-Lokalisierungs-Audit

Dieser temporäre Audit prüft `Dialog*.7z` ausschließlich auf technische Metadaten:

- Anzahl der `.conversation`- und `.stringtable`-Dateien
- Prüfsummen und Dateigrößen
- Übereinstimmung der 20 in Serins Testsave referenzierten Dialoge
- gemeinsame Verfügbarkeit von Dialoggraph und lokalisiertem Text

Der Workflow veröffentlicht keine Originaldialogtexte. Vollständige Spielressourcen sollen nach erfolgreicher lokaler Indexierung wieder aus dem öffentlichen Repository entfernt werden.
