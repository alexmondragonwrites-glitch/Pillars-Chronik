# Lokale Spielressourcen

Wächterfeder liest Dialoggraphen und lokalisierte Texte direkt aus einer lokalen *Pillars of Eternity*-Installation. Vollständige Spielressourcen, Savegames und erzeugte Dialogreports werden nicht im öffentlichen Repository gespeichert.

Lokale Pfade liegen in `.wachterfeder/config.json`; der Ordner ist per `.gitignore` ausgeschlossen.
